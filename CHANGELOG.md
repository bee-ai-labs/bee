## [0.1.0] — Foundation — 2026-07-03

The first public milestone (M0): the scaffolding that makes Bee credible, navigable, and
contributable.

### Added

- **Repository structure** — `docs/`, `examples/`, `architectures/`, `templates/`, `guides/`,
  `tutorials/`, `datasets/`, `benchmarks/`, `assets/`, `scripts/`.
- **README** — mission, features, structure, quick start, learning paths, FAQ.
- **Documentation website** — MkDocs Material config with dark mode, search, Mermaid diagrams,
  and difficulty tags.
- **Dual license** — CC-BY-4.0 for content, MIT for code.
- **Community files** — `CONTRIBUTING`, `CODE_OF_CONDUCT`, `SECURITY`, `SUPPORT`, `ROADMAP`,
  `GOVERNANCE`, `GOOD_FIRST_ISSUES`, `PROJECT_BOARD`, `DISCUSSIONS`.
- **GitHub automation** — issue/PR templates, CI workflows (format, lint, spell-check,
  link-check, docs build, stale-bot), `CODEOWNERS`, Dependabot, labels, funding config.
- **Tooling configs** — `.editorconfig`, Prettier, markdownlint, cspell + project dictionary.
